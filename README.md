# 📺 Netflix Movie Data Analysis Project

## 📌 Overview
This project performs exploratory data analysis (EDA) on a movie dataset to uncover insights about genres, popularity trends, and movie releases over time.

## 🔧 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🧹 Data Cleaning
- Handled missing values
- Converted data types
- Extracted release year
- Removed unnecessary columns

## 🔄 Feature Engineering
- Categorized movie ratings into 4 groups
- Split multiple genres into individual rows

## 📊 Key Insights
- Drama is the most frequent genre
- Adventure & Fantasy movies have highest popularity
- 2020 had the highest number of movie releases
- Spider-Man: No Way Home is the most popular movie

## 📈 Visualizations
- Genre distribution
- Popularity trends
- Year-wise movie releases
- Top 10 popular movies
